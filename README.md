# Python Projects for Beginners

Welcome to this collection of beginner-friendly Python projects!

## Overview
This repository contains simple Python projects designed to help beginners learn programming concepts through hands-on practice.


## Projects
- **Dice Rolling Game** : A simple game that simulates rolling dice.

## Getting Started

### Prerequisites
- Python 3.7 or higher
- Basic text editor or IDE (VS Code, PyCharm, etc.)

### Installation
```bash
git clone https://github.com/fannypil/Python-Projects-for-Beginners
cd Python_Projects_for_Beginners
```

### Running a Project
To run the Dice Rolling Game, use the following command:
```bash
python dice_rolling_game.py
```
To run the Number Guessing Game, use:
```bash
python number_guessing_game.py
```
